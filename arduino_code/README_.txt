1- Include the following link to the Board Manager in Arduino IDE
	* Arduino IDE: "File -> Preference -> Additional Board Manager URLs"
	Link: https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/arduino/package_m5stack_index.json

2- Navigate to: "Tool -> Board -> Board Manager..."
	* Look for the "M5core2" board and install it.

3- Install libraries
	* navigate to: "Tool -> Library Manager"
		Look for "M5core2" and install libraries

4- Restart Arduino IDE

5- Open ARduino and select the right Board
	* "Tool -> Board -> M5stack Arduino", here select the board named: "M5stack-Core2"

6. Compile and load the given source code

Detailed description given in: https://docs.m5stack.com/en/arduino/arduino_core2_development
